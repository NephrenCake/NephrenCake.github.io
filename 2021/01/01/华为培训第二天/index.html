openEuler基本命令操作

1.显示当前路径

2.显示文件或目录

ls+空格 （查看指定目录

ls 显示当前目录下的文件或目录

ls空格 查看指定目录。例：ls / 查看根目录

ls -l查看详细信息(或者ll /)

ls -a查看所有文件（包括隐藏文件

第一列，三中不同权限，对当前用户，与当前用户组，对其他用户

第二列，root属于root用户

第三列，root属于用户组

第四列，大小

第五列，时间

ls 空格 - -help(中间无空格）

按tab键补全，当不知道什么时候按tab的时候每次都试一下，但是当无论如何都不能补全的时候，应当确认是否有该文件：连按两次tab键可以列出所有关键词对应的文件或目录

3.切换目录

cd+空格+目录

切换到家目录

  • cd ~
  • cd /root
  • cd

cd ..返回上一目录

mkdir+空格+目录名 创建目录

蓝色代表文件夹

创建多级目录（当多级目录中间有文件夹不存在时） mkdir + 空格 + -p + 空格 + 目录

4.创建文件

touch 空格 新文件名

5.复制文件

cp+空格+源文件+空格+路径 （文件名不发生改变

cp+空格+源文件+空格+路径/新文件名 （文件名改变

复制文件夹cp -r test /mnt/

当文件复制不加类型时，其也单纯只是文件名的差异

6.移动命令

mv+空格+源文件+空格+路径

mv+空格+源文件+空格+路径/文件

mv+空格+路径/源文件+空格+路径/文件

用mv进行重命名

mv+空格+源文件+空格+新文件名

7.删除文件

rm 空格 文件

rm 空格 空目录

rm 空格 -r 目录 （删除非空文件夹，无论空或非空都需要确认

rm 空格 -rf 目录 （删除非空文件夹，无需确认

8.创建软连接

相当于快捷方式

当源文件删除时，该软连接失效

步骤 11 创建huawei.txt的硬链接到/mnt，并命名为huawei1.txt。

9.创建硬链接

给一个文件多路读取，原路删除不影响其他

软连接要用绝对路径

步骤 12 创建huawei.txt的软链接到/mnt，并命名为huawei2.txt。

10.查看命令

cat 空格 文件

  • 选中复制，右键粘贴，上下切换命令

head 空格 -n 空格 行数 空格 文件 （从头

tail 空格 -n 空格 行数 空格 文件 （从尾

  • mv test/huawei1.txt . （把文件移动到当前目录

12.vim文本编辑器

vi huawei.txt （如果不存在则创建并编辑

i （从当前位置进入编辑模式

o （从下一行开始编辑

esc （退出编辑模式

冒号wq （保存并退出

u （在非编辑模式撤销上一操作

冒号set nu （显示行号

冒号+行数 （跳转到指定行

G （非编辑模式中跳转到最后一行

gg （跳转到第一行

yy （复制光标所在行

P （在光标上一行粘贴复制

p （在光标下一行粘贴复制

dd （删除某一行
